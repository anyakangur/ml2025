# Нейронные сети и компьютерное зрение (CV)

### Задача 1:

Ознакомьтесь с соревнованием на Kaggle Dirty vs Cleaned V2: https://www.kaggle.com/c/platesv2. 

Мытьё тарелок — скучное занятие. К счастью, половина тарелок уже чистая. Обучите классификатор для определения чистых тарелок, чтобы сэкономить время для нового курса по машинному обучению!

Это соревнование по few-shot learning (обучению с малым количеством примеров). В тренировочном наборе всего 20 чистых и 20 грязных тарелок, а в тестовом — сотни тарелок.

НЕ ИСПОЛЬЗУЙТЕ какие-либо данные для обучения кроме тех, что есть в train выборке. Очевидно, что данных для обучения мало и что увеличение выборки сильно улучшит качество модели, но это будет совсем другая задача.

Используйте все что было изучено на семинаре.
